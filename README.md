# broodwar.live brand

Brand assets for the broodwar.live project.

## Brandmark

Typographic wordmark in monospace. The `.live` TLD is set in amber to distinguish it from the main word.

### Wordmark

Full `broodwar.live` logotype for headers, banners, and documentation.

| Variant | File | Use |
|---|---|---|
| Light text | [broodwar-live-light.svg](wordmark/broodwar-live-light.svg) | Dark backgrounds |
| Dark text | [broodwar-live-dark.svg](wordmark/broodwar-live-dark.svg) | Light backgrounds |
| On dark bg | [broodwar-live-on-dark.svg](wordmark/broodwar-live-on-dark.svg) | Standalone, dark |
| On light bg | [broodwar-live-on-light.svg](wordmark/broodwar-live-on-light.svg) | Standalone, light |

### Square Mark

Compact `bw` monogram with amber dot for avatars, app icons, and social.

| Variant | File | Use |

| Light | [bw-square-light.svg](square/bw-square-light.svg) | Light contexts |

## Colors

| Name | Hex | Use |
|---|---|---|
| Zinc 950 | `#18181B` | Primary text, dark backgrounds |
| Zinc 200 | `#E4E4E7` | Light text on dark |

| Amber 500 | `#F59E0B` | `.live` accent (on dark) |
| Amber 600 | `#D97706` | `.live` accent (on light) |

## Typography

The brandmark uses a monospace stack, reflecting the data-driven and technical nature of the project:

```
'SF Mono', 'Fira Code', 'JetBrains Mono', 'Cascadia Code', monospace
```

- **broodwar** — bold weight (700)
- **.live** — regular weight (400) in wordmark, bold in square mark
